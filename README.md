Objetivos 

• Desenvolver uma animação básica utilizando o elemento <canvas> e 
Javascript. 

• Explorar a manipulação de objetos gráficos que se movimentam e mudam de 
cor ao longo do tempo. 

• Introduzir conceitos de interatividade aplicados a elementos gráficos em 
páginas web. 

Solução Digital 


• Ferramenta utilizada: Visual Studio Code (VS Code) 

• Descrição: O VS Code é um editor de código leve e multiplataforma que oferece 
suporte a diversas linguagens de programação. Ele permite criar, editar e 
executar projetos de forma prática, com recursos de extensões e depuração 
integrados. 

• Tecnologias: HTML, CSS e JavaScript 

Procedimento / Atividade 
Atividade proposta: 

Criar uma página web que contenha uma animação simples 
utilizando o elemento <canvas> do HTML5. 

Etapas realizadas: 

• Configuração do projeto: criação de um arquivo HTML com o elemento 
<canvas> e um arquivo JavaScript para a lógica da animação. 

• Configuração inicial: definição do tamanho do <canvas> e obtenção do 
contexto 2D em JavaScript. 

• Desenho de objetos: implementação de formas gráficas (círculos, retângulos) 
utilizando métodos como ctx.beginPath(), ctx.arc() e ctx.fillRect(). 

• Animação básica: utilização do requestAnimationFrame para criar um loop 
de animação contínuo. 

• Atualização dinâmica: alteração das propriedades dos objetos (posição, cor, 
tamanho) para gerar efeitos visuais. 

• Interatividade opcional: inclusão de eventos de mouse para modificar os 
objetos (ex.: clique para mudar cor).
